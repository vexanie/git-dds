Perfect! Now you're probably wondering how to get your teammate's changes on your computer. Let the sharing begin!

Follow these steps:

  1. Open SourceTree.
  2. Notice that the index.html file has some changes to it: the changes you just made!
  3. Commit your changes. Make sure to type in a commit message to describe what update you are making. Example: I updated my bio on the page!
  4. Refresh the webpage.

Hmmm...nothing changed; Your teammate's stuff is not there, and if you check your repo on github.com, your commit is not there! The reason is because you haven't pushed your changes to github yet.

To make it work, you must both PUSH your code.

Uh oh...one of your probably got an error. That's because you can't PUSH if someone has pushed code before you do. Don't worry, there's an easy solution:

  1. Press the PULL button, then press OK.
  2. Now try pushing your code.

What PULL does is get all the latest changes from github.

You should now both be in sync! Refresh the browser, and your teammate's code should be there :)

Exercise 3

I should be red and blinking! If I'm not, then something is wrong.

Admire the text above. It's beautiful, isn't it?

Javascript is what's making the text blink. You can see the code inside index.html. Can you find it?

CSS is what's making the text red. It is also in index.html. Can you find it?

Sometimes it's good to have your css and javascript in the same files as your html; normally it's when you're just starting a project. But quickly your CSS and javascript will become big, and it will create a MASSIVE html file that is pretty confusing.

That's why it's best to split them into their own files.

Team Member 1

  1. Take all the CSS and put it in main.css inside the css folder. Don't forget to save your index.html.
  2. Refresh your page. Make sure the text is still red. If it's not, you need to link the main.css file to your html.
  3. Commit & push your changes to github.

Team Member 2

  1. Take all the javascript and put it in main.js inside the js folder. Don't forget to save your index.html.
  2. Refresh your page. Make sure the text is still blinking. If it's not, you need to link the main.js file to your html.
  3. Commit & push your changes to github.

Before moving on, you must both be in sync. Make sure to press "PULL" so you have the latest changes on your computer. The javascript & CSS should be out of your index.html
